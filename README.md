Electronic Inventory Management System

Project Description
====================
This project is an Electronic Inventory Management System developed using
Python and Tkinter. It allows staff to manage electronic items by adding,
updating, deleting, searching, and saving inventory records.

Requirements
============
- Python 3.x
- Pillow library

Install Pillow using:
pip install pillow

Files Required
==============
- ELECTRONIC INVENTORY MANAGEMENT.py
- inventory.json (generated automatically after saving data)
- LogoCompany.png
- Background.jpeg
- LOGO LOGIN.webp

How to Run
==========
1. Make sure all project files are in the same folder.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

   ELECTRONIC INVENTORY MANAGEMENT.py

Main Features
=============
- Staff login authentication
- Add new inventory items
- Update existing items
- Delete items
- Search items by Item ID
- Highlight searched records
- Low stock alert
- Save and load data using JSON
